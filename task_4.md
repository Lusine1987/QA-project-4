# Black-box тестирование приложения "Найди и иди"

## Чек-лист проверок

### 1. Поле ввода названия магазина (SC21-01)

**Позитивные тесты:**
- [ ] Ввод 2 символов (минимальное количество)
- [ ] Ввод более 2 символов
- [ ] Ввод текста с пробелами
- [ ] Ввод текста с различными символами (буквы, цифры, спецсимволы)
- [ ] Проверка отображения плейсхолдера при пустом поле
- [ ] Исчезновение плейсхолдера при начале ввода
- [ ] Проверка сохранения истории предыдущих поисков
- [ ] Выбор значения из истории поисков

**Негативные тесты:**
- [ ] Ввод 1 символа (ниже минимального)
- [ ] Ввод только пробелов
- [ ] Попытка ввода при отключенной клавиатуре

### 2. Поле выбора типа магазина (SC21-02)

**Позитивные тесты:**
- [ ] Открытие выпадающего меню при тапе на поле
- [ ] Выбор значения "Продуктовые магазины"
- [ ] Выбор значения "Магазины стройматериалов"
- [ ] Выбор значения "Аптеки"
- [ ] Отображение выбранного значения в поле после выбора
- [ ] Отображение плейсхолдера "Выберите тип магазина" при отсутствии выбора

**Негативные тесты:**
- [ ] Попытка закрыть выпадающее меню без выбора значения
- [ ] Поведение при быстром последовательном выборе разных значений

### 3. Кнопка "Поиск" (SC21-03)

**Позитивные тесты:**
- [ ] Активность кнопки при заполнении только поля названия магазина
- [ ] Активность кнопки при выборе только типа магазина
- [ ] Активность кнопки при заполнении обоих полей
- [ ] Отображение результатов поиска в виде флажков на карте
- [ ] Возможность скроллинга карты
- [ ] Возможность зуммирования карты (стандартные жесты)

**Негативные тесты:**
- [ ] Неактивность кнопки при пустых полях
- [ ] Поведение при отсутствии результатов поиска
- [ ] Поведение при потере соединения во время поиска

### 4. Кнопка "Построить маршрут" (SC21-04)

**Позитивные тесты:**
- [ ] Активность кнопки при выборе флажка найденного магазина по названию
- [ ] Активность кнопки при выборе типа магазина (поиск ближайшего)
- [ ] Отображение иконки изогнутой стрелки
- [ ] Отображение надписи "Построить маршрут"
- [ ] Белый цвет фона (#fff) в обычном состоянии
- [ ] Смена цвета фона на серый (#586d73) при нажатии
- [ ] Смена названия на "Отменить маршрут" при нажатии

**Негативные тесты:**
- [ ] Неактивность кнопки без выбора флажка или типа магазина
- [ ] Неактивность кнопки при выборе флажка без указания названия

### 5. Область карты (SC21-05)

**Позитивные тесты:**
- [ ] Занимает половину экрана телефона
- [ ] Построение маршрута при выполнении условий из SC21-04
- [ ] Отмена построенного маршрута кнопкой "Отмена"

**Негативные тесты:**
- [ ] Поведение при попытке построить маршрут без разрешения геолокации
- [ ] Поведение при изменении местоположения во время построенного маршрута

### 6. Общая функциональность (SC21-06)

**Позитивные тесты:**
- [ ] Отображение незаконченного маршрута при повторном открытии приложения
- [ ] Сохранение состояния приложения между сессиями

**Негативные тесты:**
- [ ] Поведение при принудительном закрытии приложения
- [ ] Поведение при перезагрузке устройства с активным маршрутом

### 7. Интеграционные тесты

**Сценарии использования:**
- [ ] Поиск магазина по названию → выбор флажка → построение маршрута
- [ ] Выбор типа магазина → поиск → построение маршрута до ближайшего
- [ ] Построение маршрута → отмена → новый поиск
- [ ] Использование истории поисков для быстрого доступа
- [ ] Работа с картой (зум, скролл) во время активного маршрута

**Пограничные случаи:**
- [ ] Одновременное заполнение обоих полей и приоритет поиска
- [ ] Поведение при быстром последовательном нажатии кнопок
- [ ] Работа приложения в фоновом режиме и восстановление состояния
- [ ] Обработка поворота экрана с активным маршрутом

### 8. Тесты удобства использования

**UI/UX проверки:**
- [ ] Соответствие макету
- [ ] Читаемость текста на карте
- [ ] Достаточный размер интерактивных элементов для нажатия
- [ ] Логичная последовательность действий
- [ ] Интуитивно понятные иконки и обозначения
